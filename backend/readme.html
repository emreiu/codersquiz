CrudServer Installation Instructions

Intro

CrudServer is a Node application using Express to create a web server. The resources you create are stored in JSON files, one file per entity. Basically it's all JavaScript and JSON. In order to run the server please follow the steps outlined below. Once you have finished these steps you can access the server documentation at http://localhost:3000

Installation steps

  1. Open the folder "app" in your IDE (VS Code, WebStorm etc.)
  2. Open up the console (make sure it is not set to PowerShell).
  3. Enter the command npm install. This will install all required node modules.
  4. Once this is finished enter the command node app. This will start the server.
  5. Make sure you don't close your IDE, otherwise the server will shut down.
